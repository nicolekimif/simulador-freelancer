# 💻 Simulador de Orçamento Freelancer

## 📌 Sobre o Projeto
O **Simulador de Orçamento Freelancer** é uma aplicação web que permite calcular de forma simples o valor estimado de serviços de tecnologia.

O usuário pode selecionar diferentes serviços de desenvolvimento web e acompanhar automaticamente o valor total do orçamento. O projeto foi desenvolvido com foco em interatividade no lado do cliente (*client-side*), utilizando JavaScript para atualizar os valores dinamicamente na página.

---

## ⚙️ Funcionalidades
- Seleção de serviços de tecnologia
- Adição de serviços ao orçamento
- Lista dinâmica de pedidos
- Cálculo automático do valor total
- Navegação entre seções da página
- Links para contato via redes sociais
